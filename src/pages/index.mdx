## What is this?

[sheetforms.co](https://sheetforms.co) is a simple API to save
your `<form>` data to a [Google Spreadsheet](https://docs.google.com/spreadsheets).

Make a `POST` request with your spreadsheet's ID and your data
to `https://sheetforms.co/api` like this:

```js
async function handleSubmit() {
  await fetch("https://sheetforms.co/api", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({
      spreadsheetId: "YOUR_SPREADSHEET_ID",
      values: ["ADD", "SOME", "DATA", "HERE"],
    }),
  });
}
```
